# Playwright Automation Demo

This project is a simple demo created to practice and learn Playwright automation testing.

The main goal of this project is to understand how to write clean and basic end-to-end tests, run them in the browser, and observe how Playwright interacts with the UI step by step.

## Purpose

This demo was created for learning purposes, focusing on:

- Setting up Playwright in a project
- Writing simple automation tests
- Running tests in headed mode to watch the browser actions
- Practicing form filling and UI interactions
- Using Cursor AI as a learning and development assistant

## Tools Used

- Playwright
- TypeScript
- Cursor AI
- Node.js

## How to Install

```bash
npm install
````

## How to Run the Tests

Run tests normally:

```bash
npx playwright test
```

Run tests and watch them in the browser:

```bash
npx playwright test --headed
```

Run tests using Playwright UI Mode:

```bash
npx playwright test --ui
```

## Test Report

After running the tests, you can view the report using:

```bash
npx playwright show-report
```
